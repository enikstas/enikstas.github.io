# Personal Portfolio & Data Analytics Showcase

## Project Purpose
This repository hosts the source code for the personal professional portfolio of **Stanislav Enikeev**, functioning as an independent Analyst-Developer showcase. The primary target audience consists of US-based small to medium business owners, operational managers, and stakeholders seeking custom data automation, legacy system integrations, and interactive reporting solutions.

The website serves a dual strategic purpose:
1. **Technical Demonstration:** Showcasing hands-on expertise in front-end data visualization (HTML5, CSS3 Custom Properties, JavaScript ES6, and interactive chart mocking), responsive layout architecture, and client-side security practices.
2. **Business Inbound Channel:** Acting as a professional digital calling card allowing potential clients to review operational solution concepts (consisting of data pipeline designs, warehouse architectures, and inventory tracking logic) and securely establish business communication.

## Core Architecture & Technical Highlights
* **Zero-Backend Architecture:** Built entirely as a lightweight, static web resource optimized for instant loading speeds, high SEO indexing standards, and mobile-first responsiveness.
* **Anti-Scraping Mitigation (Client-Side Obfuscation):** Utilizes an event-driven execution loop that reconstructs contact data in-memory strictly upon explicit user interaction, successfully mitigating high-volume automated HTML/JS regex parsing engines.
* **Interactive Visual Layer:** Features dynamic interactive tab switching mimicking modern analytics dashboard experiences (Revenue, Channels, Products layout transitions) without relying on heavy third-party framework overhead.
